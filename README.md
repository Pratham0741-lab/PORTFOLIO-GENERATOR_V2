To use this Model you must have an Google Gemini API key which you can get here: https://aistudio.google.com/api-keys
after this create one API key and make a file named ".env" in the folder where you have kept the app.py
now in the .env file <<GENAI_API_KEY="PASTE YOUR GEMINI API KEY">> paste this line and paste your gemini key without quotes for example GENAI_API_KEY=AIz.....
now open app.py file and in terminal type --> python app.py
then you will get a local host link like http://127.0.0.1:5000
open it and use the AI PORTFOLIO GENERATOR
